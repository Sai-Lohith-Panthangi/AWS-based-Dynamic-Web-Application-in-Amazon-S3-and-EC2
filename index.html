By streamlining the coding experience, the tool empowers developers to navigate extensive codebases more effectively, reducing the time spent searching for specific functions, variables, or classes. Breast Cancer Detection Using Meta-Learning And ANN

In this Project, we have analyzed the various methods used by the researchers, their advantages and limitations and we have tried to use, various supervised and unsupervised learning models like Random Forest, KNN, SVM, Logistic Regression, Ada Boost, Perceptron, and feature selection. The data set used for this is the Wisconsin Breast Cancer Data set which has 569 instances of breast cancer patients. Metrics like Accuracy, Precision, Recall, F1-Score, and ROC-curve are used to judge the performance of the model.
